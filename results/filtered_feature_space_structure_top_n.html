15  Structure and Overlap of Top-N Variance Probe Sets

Preliminary diagnostic comparison of comparison-specific and latent-space feature sets

15.1 Overview

This section presents a preliminary diagnostic analysis of top-N variance-selected probe sets. It is intended to complement, but not replace, the existing filtered feature-space structure section based on limma-derived comparison-specific probe sets.

The existing filtered feature-space chapter uses UpSet and bar-plot summaries to describe overlap among filtered probe sets across cancer categories. The present section follows that conceptual role, but addresses a different question: whether the comparison-specific top-3000 variance probe sets used in the analysis pipeline overlap with the global top-3000 variance probe set used for latent-space modeling.

The diagnostic is currently restricted to the hu35ksuba chip and is valid only when the analysis filtered-probe object was generated with:

parameter required_value
filter_method variance
variance_selection_mode top_n
analysis_top_n 3000

Under this configuration, each normal/tumor comparison contributes a fixed-size set of 3000 high-variance probes. These comparison-specific sets are compared to the global top-3000 variance feature set used as input for the latent-space workflow.

This analysis should be interpreted as a feature-space alignment diagnostic, not as a direct biological complexity result. It asks whether two downstream analyses are operating on overlapping molecular feature spaces.

15.3 Overall overlap

n_comparisons min_overlap_fraction median_overlap_fraction mean_overlap_fraction max_overlap_fraction min_jaccard median_jaccard mean_jaccard max_jaccard
15 0.582 0.614 0.62 0.667 0.411 0.443 0.45 0.5

Across comparisons, the comparison-specific top-3000 variance probe sets overlap substantially with the global latent-space top-3000 variance feature set. In the preliminary results, overlap fractions range from approximately 0.58 to 0.67, with Jaccard similarity values of approximately 0.41 to 0.50.

Because both feature sets contain 3000 probes, overlap_fraction_analysis and overlap_fraction_latent are identical in this diagnostic. The Jaccard index provides a more conservative measure of set similarity because it uses the size of the union as its denominator.

These results indicate that the latent-space and comparison-specific analysis workflows share a large high-variance probe backbone, while retaining a substantial comparison-specific component.

Group-level variation is informative because the latent-space feature set is global, whereas the analysis feature sets are comparison-specific. Higher overlap suggests that a disease group’s local variance structure is more strongly represented in the global high-variance feature space. Lower overlap suggests a larger comparison-specific component not fully captured by the global latent feature set.

This should not be interpreted as a ranking of biological importance or disease complexity. Rather, it describes how strongly each disease category aligns with the global variance structure used for latent-space modeling.

15.10 Interpretation and limitations

The preliminary top-N diagnostic supports three cautious conclusions.

First, the latent-space and comparison-specific complexity/entropy analyses are not operating on unrelated feature spaces. Most comparisons share roughly 60% of their selected probes with the global latent-space feature set.

Second, the feature spaces are not identical. Approximately one-third to two-fifths of probes differ between the global latent feature set and each comparison-specific analysis feature set. Therefore, agreement or disagreement between latent-space geometry and complexity/entropy metrics should be interpreted with this feature-space difference in mind.

Third, group-level variation may itself be informative as a diagnostic observation. Differences among carcinomas, blastomas, lymphomas, and leukemias may reflect variation in how strongly each disease category contributes to, or diverges from, the global high-variance structure used for latent-space modeling.

These results should not be overinterpreted as direct evidence of differences in biological complexity. They characterize the feature spaces within which complexity, entropy, and latent-space properties are evaluated.

15.11 Relationship to limma-based feature-space structure

The existing filtered feature-space structure section describes overlap among limma-derived comparison-specific probe sets. The present section instead describes overlap between comparison-specific top-N variance probe sets and a global top-N variance probe set used for latent-space modeling.

These are complementary diagnostics. The limma-based analysis emphasizes differential expression structure, whereas the top-N variance analysis emphasizes high-variance transcriptomic structure. Future reporting should explicitly preserve this distinction.

15.12 Future extensions

Future versions of this diagnostic should compare additional feature-selection schemes, including:

  • limma-derived filtered probe sets;
  • variance-threshold filtered probe sets;
  • matched top-N differential-expression probe sets;
  • platform-specific latent feature sets for chips beyond hu35ksuba;
  • pairwise comparisons between local top-N, global top-N, and differential-expression feature spaces.

These extensions would allow direct evaluation of how sensitive complexity measures, entropy measures, and latent-space geometry are to the feature-selection strategy.
